Lesson 9: CSS Properties for Text Formatting

CSS Text Formatting Properties

In this lesson, we will learn about various CSS properties that are used to format text. These properties help you control the appearance of text on a web page. Here are some key properties:

1. font-family

This property specifies the font of the text. You can use system fonts or web fonts.

p {
  font-family: Arial, sans-serif;
}

2. font-size

Sets the size of the font. You can use units such as pixels (px), ems, or percentages.

p {
  font-size: 16px;
}

3. font-weight

Defines the thickness of the font. Values can be normal, bold, bolder, or numbers from 100 to 900.

p {
  font-weight: bold;
}

4. text-align

Aligns the text horizontally within its container. Values can be left, right, center, or justify.

p {
  text-align: center;
}

5. line-height

Sets the distance between lines of text. It can be a number, length, or percentage.

p {
  line-height: 1.5;
}

6. text-transform

Controls the capitalization of text. Options include uppercase, lowercase, and capitalize.

p {
  text-transform: uppercase;
}

7. letter-spacing

Adjusts the space between letters.

p {
  letter-spacing: 2px;
}
