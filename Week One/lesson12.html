Lesson 12: Complex CSS Selectors (Pseudo-classes and Pseudo-elements)

Complex CSS Selectors: Pseudo-classes and Pseudo-elements

In this lesson, we’ll explore more advanced CSS selectors: pseudo-classes and pseudo-elements. These selectors allow you to style elements based on their state or to style specific parts of an element.

1. Pseudo-classes

Pseudo-classes target elements based on their state or their position within the DOM. They are written with a colon (:) before the class name.

  • :hover: Targets an element when the user hovers over it with a cursor.
  • :focus: Targets an element when it is focused, such as when a user clicks on an input field.
  • :nth-child(): Targets an element based on its position among its siblings. For example, :nth-child(2) targets the second child.
  • :not(): Excludes elements that match a specific selector. For example, :not(.highlight) targets all elements except those with the class "highlight".
/* Example of using pseudo-classes */
button:hover {
  background-color: #6d9cce;
}

input:focus {
  border-color: #0758ad;
}

li:nth-child(odd) {
  background-color: #f4f4f4;
}

2. Pseudo-elements

Pseudo-elements allow you to style specific parts of an element, such as the first letter or the first line of a paragraph. They are written with two colons (::) before the element name.

  • ::before: Inserts content before the element’s actual content.
  • ::after: Inserts content after the element’s actual content.
  • ::first-letter: Styles the first letter of an element.
  • ::first-line: Styles the first line of an element.
/* Example of using pseudo-elements */
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

p::before {
  content: "Note: ";
  color: red;
  font-weight: bold;
}

3. Combining Pseudo-classes and Pseudo-elements

You can combine pseudo-classes and pseudo-elements for more powerful selectors:

/* Example of combining pseudo-classes and pseudo-elements */
a:hover::after {
  content: " →";
  color: #0758ad;
}
