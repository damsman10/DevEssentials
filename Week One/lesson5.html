Lesson 5: Images, Videos, and Audios in HTML

Introduction to Multimedia Elements

HTML allows you to embed images, videos, and audios directly into your webpage. Let's explore how to use these elements effectively.

Images (<img>)

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image, and the alt attribute provides alternative text for accessibility.

<img src="image.jpg" alt="A description of the image">

Videos (<video>)

The <video> tag is used to embed a video on a webpage. You can also specify controls like play, pause, and volume using the controls attribute.

<video src="video.mp4" controls>
  Your browser does not support the video tag.
</video>

Audios (<audio>)

The <audio> tag is used to embed audio files. Like the <video> tag, you can include controls with the controls attribute.

<audio src="audio.mp3" controls>
  Your browser does not support the audio tag.
</audio>
