Lesson 11: Selectors and Grouping by ID

Selectors and Grouping by ID

In this lesson, we'll dive into CSS selectors, focusing on how to select elements by their ID, as well as grouping multiple selectors together to apply the same styles.

1. Understanding Selectors

CSS selectors are patterns used to select elements on a web page for styling. The most common types of selectors include:

  • Element Selector: Selects elements by their HTML tag name. For example, p will select all paragraph elements.
  • ID Selector: Selects a single element by its unique ID, using the # symbol. For example, #header selects the element with the ID "header".
  • Class Selector: Selects elements by their class name, using the . symbol. For example, .button selects all elements with the class "button".

2. Grouping Selectors

Sometimes, you might want to apply the same style to multiple elements. Instead of repeating the CSS rule for each selector, you can group them together using a comma:

/* Example of grouping selectors */
#header, .nav, .footer {
  background-color: #0758ad;
  color: white;
}

3. ID vs Class

It's important to understand the difference between IDs and classes in CSS:

  • ID: Should be unique within the page. Use it to style a single element.
  • Class: Can be reused across multiple elements. Use it to apply the same style to multiple elements.
/* Example of using ID and class selectors */
#header {
  font-size: 24px;
  text-align: center;
}

.button {
  padding: 10px 20px;
  background-color: #ff5733;
  color: white;
}
