Lesson 3: HTML Attributes

What are HTML Attributes?

HTML attributes provide additional information about elements. They are always specified in the opening tag and usually come in name/value pairs like name="value".

Common HTML Attributes

  • href: Specifies the URL of a link.
  • src: Specifies the source of an image or script.
  • alt: Provides alternative text for an image.
  • title: Adds a tooltip text when the mouse hovers over the element.
  • style: Applies inline CSS styles.

Example: Using Attributes in HTML

Here’s an example of how to use these attributes:

<a href="https://example.com" title="Visit Example">Example Link</a>
<img src="image.jpg" alt="A descriptive image">
