Lesson 4: HTML Lists

Introduction to HTML Lists

HTML provides several ways to create lists. The most common are ordered lists, unordered lists, and description lists.

Ordered Lists (<ol>)

Ordered lists are used when the order of items matters. Each item is automatically numbered.

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

This code produces:

  1. Item 1
  2. Item 2
  3. Item 3

Unordered Lists (<ul>)

Unordered lists are used when the order of items doesn’t matter. Each item is typically marked with a bullet point.

<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>

This code produces:

  • Item A
  • Item B
  • Item C

Description Lists (<dl>)

Description lists are used to list terms and their descriptions. Each term is wrapped in a <dt> tag, and each description in a <dd> tag.

<dl>
  <dt>Term 1</dt>
  <dd>Description of Term 1</dd>
  <dt>Term 2</dt>
  <dd>Description of Term 2</dd>
</dl>

This code produces:

Term 1
Description of Term 1
Term 2
Description of Term 2
