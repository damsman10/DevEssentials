Lesson 6: Links in HTML

Introduction to Links

Links are essential in HTML for navigating between pages and websites. In this lesson, you'll learn how to create and style links effectively.

Basic Link Syntax

The <a> (anchor) tag is used to create a hyperlink in HTML. The href attribute specifies the URL of the page the link goes to.

<a href="https://www.example.com">Visit Example</a>

Opening Links in a New Tab

To open a link in a new tab, use the target="_blank" attribute.

<a href="https://www.example.com" target="_blank">Visit Example</a>

Email Links

To create a link that opens the user's email client, use the mailto: protocol.

<a href="mailto:someone@example.com">Send Email</a>

Linking to Sections Within a Page

To link to a specific section within the same page, use an ID selector with the # symbol.

<a href="#section1">Go to Section 1</a>
