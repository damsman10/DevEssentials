Lesson 8: Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the presentation of a web page, including colors, layout, and fonts. It allows you to create visually appealing websites by controlling how elements are displayed on the page.

Basic CSS Syntax

CSS is written as rules that target HTML elements. A rule consists of a selector and a declaration block.

selector {
  property: value;
}

Applying CSS to HTML

There are three ways to apply CSS to HTML: inline styles, internal stylesheets, and external stylesheets.

  • Inline Styles: Directly in the HTML element using the style attribute.
  • Internal Stylesheet: Within the <style> tags in the <head> section of the HTML document.
  • External Stylesheet: Linking to a separate CSS file using the <link> tag.
