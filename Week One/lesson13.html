Lesson 13: CSS Display and Positioning

Understanding CSS Display Property

The display property in CSS determines how an element is displayed on the web page. The most common values for the display property are:

  • block: The element takes up the full width available, with a new line before and after it.
  • inline: The element takes up only as much width as necessary, without starting a new line.
  • inline-block: The element is like inline, but you can set width and height.
  • none: The element is completely removed from the document flow.
/* Example of using display property */
div {
  display: block;
}

span {
  display: inline;
}

CSS Positioning

CSS positioning allows you to control the placement of elements on your webpage. The main positioning types are:

  • static: This is the default position; elements are placed according to the normal document flow.
  • relative: The element is positioned relative to its normal position, using top, right, bottom, and left properties.
  • absolute: The element is positioned relative to its closest positioned ancestor (not static). If none, it uses the document body.
  • fixed: The element is positioned relative to the browser window, staying in the same position even when scrolling.
  • sticky: The element toggles between relative and fixed positioning depending on the scroll position.
/* Example of using positioning */
.relative-box {
  position: relative;
  top: 10px;
}

.absolute-box {
  position: absolute;
  top: 50px;
  left: 100px;
}

.fixed-box {
  position: fixed;
  bottom: 10px;
  right: 10px;
}
