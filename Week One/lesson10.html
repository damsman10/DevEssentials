Lesson 10: CSS Colors and Fonts

CSS Colors and Fonts

In this lesson, we will explore how to use CSS to style colors and fonts on your web page. We'll learn about color formats, font properties, and how to apply them effectively.

1. Colors in CSS

Colors in CSS can be specified using different formats:

  • Hexadecimal: A 6-digit code starting with #, e.g., #ff5733.
  • RGB: Defines colors using Red, Green, and Blue values, e.g., rgb(255, 87, 51).
  • RGBA: Similar to RGB but includes an Alpha (opacity) value, e.g., rgba(255, 87, 51, 0.5).
  • Named Colors: CSS provides predefined color names, e.g., red, blue, green.
/* Example of using colors */
body {
  background-color: #ff5733;
  color: white;
}

p {
  color: rgb(255, 255, 255);
}

a {
  color: rgba(255, 255, 255, 0.8);
}

2. Fonts in CSS

CSS offers various properties to style fonts:

  • font-family: Sets the font type. You can specify multiple fonts as fallbacks.
  • font-size: Controls the size of the font. Can be set in px, em, rem, etc.
  • font-style: Defines the style of the font, such as normal, italic, or oblique.
  • font-weight: Specifies the thickness of the font, with values ranging from 100 to 900.
  • font-variant: Controls the use of small-caps.
/* Example of using fonts */
body {
  font-family: 'Arial', sans-serif;
  font-size: 16px;
  font-style: normal;
  font-weight: 400;
  font-variant: small-caps;
}
