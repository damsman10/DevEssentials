Lesson 2: HTML Text Formatting

HTML Text Formatting Tags

HTML provides several tags that allow you to format text in various ways. These include making text bold, italic, underlined, and more.

Common Text Formatting Tags

  • <b>: Bold text
  • <i>: Italic text
  • <u>: Underlined text
  • <em>: Emphasized text
  • <strong>: Important text

Example: Formatting Text in HTML

Here’s an example of how to use these tags to format text:

<p>This is <b>bold</b>, <i>italic</i>, and <u>underlined</u> text.</p>
