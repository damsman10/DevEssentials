Assignment: QR Component Design

This assignment is designed to test your understanding of HTML and CSS. You will be creating a QR code component that displays a QR code image, a title, and a short description, similar to a challenge from Frontend Mentor.

Assignment Details:

  • Create a card with a QR code image and some text below it.
  • Ensure the card is centered on the page both vertically and horizontally.
  • The card should be responsive, adjusting properly for mobile and desktop views.
  • Use Flexbox or CSS Grid for layout.
  • Follow the design provided in the reference image from Frontend Mentor.
