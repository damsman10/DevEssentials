Lesson 3: HTML Tables

What are HTML Tables?

HTML tables allow you to display data in a structured, tabular format. Tables are commonly used to present data in rows and columns.

Basic Table Elements

  • <table>: Container for the table.
  • <tr>: Table row.
  • <td>: Table data cell.
  • <th>: Table header cell.

Example: Simple HTML Table

Here is an example of a basic HTML table:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
    <td>Chicago</td>
  </tr>
</table>
