Lesson 5: CSS Grid

What is CSS Grid?

CSS Grid is a layout system that enables you to create complex and responsive web layouts with ease. It provides a two-dimensional grid-based layout, allowing for rows and columns, making it perfect for building web page layouts.

Defining a Grid Container

To start using CSS Grid, apply display: grid; to a container element. This container will serve as a grid container for the child elements (grid items).

.grid-container {
  display: grid;
}

Setting Up Columns and Rows

Use grid-template-columns and grid-template-rows to define the number and size of columns and rows in the grid container:

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: auto;
}

In this example, the container has three columns of equal width, where 1fr (fractional unit) divides space equally.

Grid Gap

Add spacing between grid items using grid-gap (or gap):

.grid-container {
  gap: 10px;
}

Placing Grid Items

Position grid items using properties like grid-column and grid-row to span columns and rows:

.grid-item {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
}

This example makes a grid item span from the first to the third column and from the first to the second row.
