Lesson 8: CSS Variables

What are CSS Variables?

CSS variables, also known as custom properties, allow you to define reusable values in your CSS that you can apply throughout your stylesheet. This can help make your code more manageable and adaptable to changes.

How to Define CSS Variables

To define a CSS variable, use the --variable-name syntax inside a CSS selector, often in the :root selector to make them global:

:root {
  --primary-color: #0758ad;
  --secondary-color: #6d9cce;
  --font-size-large: 18px;
}

Using CSS Variables

To use a CSS variable, reference it with the var() function:

body {
  color: var(--primary-color);
  font-size: var(--font-size-large);
}

Overriding CSS Variables

You can override variables within specific elements to create different themes or adapt styles in various contexts:

.dark-theme {
  --primary-color: #333;
}

Applying var(--primary-color) inside .dark-theme will now use #333 instead of the default color.
