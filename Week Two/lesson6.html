Lesson 6: CSS Responsive Design

What is Responsive Design?

Responsive design is an approach that allows web pages to adapt to different screen sizes, ensuring they look good on all devices, from mobile phones to desktops.

Core Concepts of Responsive Design

  • Media Queries: CSS rules that apply styles based on device characteristics, like screen width.
  • Flexible Units: Units like percentages, em, and rem that allow layouts to adjust to screen size.
  • Responsive Images: Images that scale based on the screen size to avoid layout issues on smaller screens.

Example: Using Media Queries

Below is an example of how to use media queries to change styles based on screen width:

/* Default styles for large screens */
.container {
  display: flex;
  flex-direction: row;
  padding: 20px;
}

/* Styles for screens smaller than 600px */
@media (max-width: 600px) {
  .container {
    flex-direction: column;
    padding: 10px;
  }
}
