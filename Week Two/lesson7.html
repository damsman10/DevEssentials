Lesson 7: CSS Media Queries

What Are Media Queries?

Media queries are a CSS feature that allows you to apply styles based on certain conditions, such as the screen's width, height, orientation, and resolution. They’re essential for creating responsive designs that adapt to different devices.

Syntax of Media Queries

A media query consists of a condition and a set of CSS rules. Here’s an example syntax:

@media (condition) {
  /* CSS rules */
}

Basic Example of Media Queries

Below is an example of a media query that applies different styles for screens narrower than 768px:

/* Default styles for larger screens */
.container {
  padding: 20px;
}

/* Styles for screens smaller than 768px */
@media (max-width: 768px) {
  .container {
    padding: 10px;
  }
}

Common Media Query Breakpoints

  • Small screens: @media (max-width: 600px)
  • Tablets: @media (max-width: 768px)
  • Desktops: @media (min-width: 1024px)
