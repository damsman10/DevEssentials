Lesson 1: Semantic HTML

What is Semantic HTML?

Semantic HTML provides meaning to web elements, enhancing accessibility and SEO. It helps browsers and search engines understand the page structure, making it more accessible and improving usability.

Semantic HTML Tags

  • <header>: Defines a header section for a document or section.
  • <nav>: Represents a section for navigation links.
  • <main>: Specifies the main content area.
  • <article>: Contains self-contained content, like a blog post.
  • <section>: Defines a section in a document.
  • <aside>: Holds content indirectly related to the main content.
  • <footer>: Defines a footer for a document or section.

Example: Semantic HTML Structure

<header>
  <h1>Welcome to My Blog</h1>
</header>

<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>

<main>
  <section>
    <article>
      <h2>Blog Post Title</h2>
      <p>This is the content of the blog post.</p>
    </article>
  </section>
</main>

<footer>
  <p>© 2024 My Blog</p>
</footer>

Practice

Create a simple page structure using the semantic HTML tags. Try structuring it as shown in the example above.
